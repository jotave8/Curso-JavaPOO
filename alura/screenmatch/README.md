# screenmatch

Simple example modeling movies.

Files
- Movie.java — movie model (title, year, rating, etc.)
- Main.java — example runner that demonstrates `Movie` usage.

How to compile
```
cd /home/jotave/Documentos/Curso-JavaPOO/alura
mkdir -p out
javac -d out screenmatch/*.java
```

How to run
```
java -cp out screenmatch.Main
```
